# 📊 Week 1 Project – Advanced EDA & Feature Engineering

## 📌 Project Overview

This project was completed as part of my **Data Science Internship at Decode Labs**.

The objective was to clean, preprocess, analyze, and extract meaningful business insights from an e-commerce dataset using Python.

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📂 Dataset Information

- Total Records: **1200**
- Total Features: **19**
- Dataset Type: E-commerce Orders

---

## 🔍 Project Workflow

- Data Loading
- Data Cleaning
- Missing Value Handling
- Duplicate Check
- Outlier Detection using IQR
- Feature Engineering
- Exploratory Data Analysis
- Business Insights
- Dataset Export


## ⚙️ Feature Engineering

Created the following new features:

- DiscountUsed
- OrderMonth
- OrderYear
- WeekendOrder
- AveragePricePerItem


## 📈 Exploratory Data Analysis

Analysis performed on:

- Product Sales
- Payment Methods
- Order Status
- Referral Sources
- Monthly Orders
- Weekend vs Weekday Orders


## 💡 Key Business Insights

- Printer recorded the highest number of orders.
- Online was the most preferred payment method.
- June had the highest order volume.
- Weekday orders were significantly higher than weekend orders.
- Feature engineering improved business analysis.
